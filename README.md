# openclaw-agent

A personal intelligent assistant system built with [OpenClaw](https://github.com/nickclyde/OpenClaw), deployed on a private server for daily knowledge management, information retrieval, and complex task orchestration.

## Overview

This project includes three core agents:

- **instone** – Knowledge Management Agent  
  Automatically maintains a vault of 4,000+ Markdown notes, performing content search, bidirectional link maintenance, and data integration to significantly reduce knowledge base upkeep effort.

- **searcher** – Multi-source Information Retrieval Agent  
  Selects optimal data sources based on user intent, queries multiple platform APIs, extracts key insights, and generates consolidated reports.

- **A管代** – Multi-Agent Coordination Agent  
  Breaks down complex, ambiguous tasks through multi-turn dialogue, decomposes them into executable subtasks, and dispatches them to specialized agents, turning unstructured demands into standardized multi-agent workflows.
